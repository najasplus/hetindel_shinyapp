About

This app is designed to analyze heterozygous and homozygous indels using .ab1 Sanger sequnce files and reference sequence in text format as a .txt file.

Instructions

Input

Create a .zip file containing your .ab1 chromatograms and corresponding reference sequence in text format (no fasta header!) as an individual .txt file with a name ending with "reference.txt"
  • The expected indel should occur 70 or more bases from the 5' and 3' ends of the sequence
  • sequences may be in forward and reverse orientation to the reference
  • the reference sequence should correspond to the sequenced PCR product
  • No non-IUPAC characters are allowed in the reference sequence
Upload your .zip archive to the app

Parameters

  • you can change the chromatogram peak detection with Signal/Noise Ratio setting. Reduce it when primary and secondary peaks are different in heigth. Increase when the noise level is high
  • check "Make Chromatograms" if you want to get the sequence chromatograms as pdf to analyze peak detection (recommended)
  • Choose 5' offset and sequence length for matching the homozygous part of sequence to the reference. Use this setting to avoid the areas rich in polymorphysms or bad sequence
  • Choose 3' offset and sequence length for matching the homozygous part of sequence to the reference
  • For more reliable allele determination it is recommeded that tested homozygous and heterozygous parts of the sequence would be in the vicinity of the indel

Output

As an output you receive a .zip file containing your original files, the chromatograms as .pdf (optionally), and two txt files.
The file *_sequences.txt contains the sequence data as a IUPAC codes, and predicted deconvolved alleles
File *_match.txt contains the information about the parameters used for the sequences processing, and for each individual sequence phase shift between the alleles and alignment of predicted alleles to the reference.
